# Mini-Project 04 (v2): Dynamic Routing (RIP)
## Objective
To enable automatic route learning between routers using a dynamic routing protocol.

## Topology
2 Routers
2 Switches
2 LANs
Router-to-router link

## Configuration Summary
RIP version 2 was configured on both routers.
Routers exchanged routing information dynamically.

## Observations
Routers automatically learned remote networks.
Routing tables updated without manual configuration.
Network adapted to changes automatically.

## Limitations Identified
RIP has slow convergence.
Not suitable for large enterprise networks.
Limited scalability.

## Conclusion
Dynamic routing eliminates manual route configuration and improves scalability,
leading to more advanced protocols like OSPF.
